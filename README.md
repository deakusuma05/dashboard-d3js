# ☁️ Superstore Cloud Explorer
### Minimalist D3.js Radial Analytics Dashboard

Superstore Cloud Explorer is a premium, minimalist analytics dashboard built with **D3.js**. It features an ethereal "Blue Sky" aesthetic with a floating UI, glassmorphism effects, and a custom interactive cloud cursor.

![Dashboard Preview](https://via.placeholder.com/1200x600?text=Superstore+Cloud+Explorer+Preview)

## ✨ Features
- **Radial Tree Map**: Interactive visualization of Superstore data (Region -> Quarter -> Month -> Category -> Sub-Category).
- **Floating UI**: Glassmorphic top-navigation and draggable info cards for a modern, clutter-free experience.
- **Dynamic Filtering**: Filter data by Region, Category, Sub-Category, and Sales Range in real-time.
- **Custom Cloud Cursor**: A playful, interactive cursor that reacts to clicks.
- **Responsive Layout**: Designed to be immersive and adaptable.
- **Performance Optimized**: Built with vanilla technologies for maximum speed and smooth animations.

## 🛠️ Tech Stack
- **D3.js v7**: Core visualization engine.
- **Vanilla CSS3**: Custom glassmorphism design system.
- **HTML5**: Semantic structure.
- **Git/GitHub**: Version control and deployment.

## 🚀 Getting Started
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/deakusuma05/dashboard-d3js.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dashboard-d3js
   ```
3. Open `index.html` in your browser.
   *Note: For the best experience (loading CSV data), it is recommended to use a local server like Live Server (VS Code) or Python's `http.server`:*
   ```bash
   python server.py
   ```
4. Visit `http://localhost:8002` in your browser.

## 🌐 Deployment
This dashboard can be easily deployed using **GitHub Pages**:
1. Push your code to GitHub.
2. Go to **Settings** > **Pages** in your repository.
3. Select the `main` branch as the source and click **Save**.
4. Your site will be live at `https://deakusuma05.github.io/dashboard-d3js/`.
